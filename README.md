# TaskMaster - Student Task Manager

A Flutter app for students to manage assignments and tasks, with full Swahili/English language support and dark/light mode.

## Features
- Login with session persistence
- Dashboard with Today, Upcoming, Overdue, Completed tabs
- Add/Edit/Delete tasks with priority levels
- Progress stats summary
- Swahili ↔ English language toggle
- Dark/Light theme toggle
- Data persisted locally via SharedPreferences

## Getting Started

```bash
git clone https://github.com/rama-dhani24/student-task-manager.git
cd student-task-manager
flutter pub get
flutter run
```
